# AWS Serverless To-Do Application

A fully serverless To-Do application built with:

- AWS Lambda
- API Gateway
- DynamoDB
- S3 Static Hosting

## Features
✔ Add tasks  
✔ Get all tasks  
✔ Delete tasks  
✔ Fully serverless backend  
✔ Modern minimalist UI  

## Technology Stack
- JavaScript (Frontend)
- HTML, CSS
- AWS Lambda (Node.js)
- AWS API Gateway (REST)
- DynamoDB (NoSQL)
- Amazon S3 (Static website hosting)
